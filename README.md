# purify.js
A JavaScript-based library that can read RSAC/RSACi and ICRA rating information, as well as PICS v1.1 HTTP headers and any RSACi/ICRA rating information embedded within them.

## What is ICRA?
ICRA was a self-applied rating system for websites. Instead of providing strict age ratings, the website would encode information about the type of content it contains - for example, language or nudity.
